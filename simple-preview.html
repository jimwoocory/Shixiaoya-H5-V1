📱 项目概述

基于 Taro 框架开发的多端板材行业应用，支持微信小程序、H5、支付宝小程序等多个平台。项目包含完整的前端展示和后端API服务。

🚀 部署方式

Docker部署 (推荐)

使用Docker Compose一键部署，包含MySQL、Redis、API服务。

cd backend
docker-compose up -d

手动部署

传统的手动部署方式，适合开发环境和自定义配置。

npm install
npm run dev
